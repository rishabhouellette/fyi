FYI UPLOADER - COMPLETE USER GUIDE
How to Use This Program
================================================================================

INTRODUCTION
================================================================================

The FYI Uploader is a comprehensive social media management platform that
allows you to manage, schedule, and optimize content across multiple social
networks from a single unified interface.

MAIN FEATURES:
- Schedule posts across Facebook, Instagram, YouTube, and more
- Manage team collaboration and approval workflows
- Track analytics and monitor performance in real-time
- Generate AI-powered content and hashtag suggestions
- Create white-label instances for agencies
- Comprehensive security with 2FA and API management


================================================================================
GETTING STARTED
================================================================================

1. INSTALLATION & SETUP
   ─────────────────────────────────────────────────────────────────────────
   
   a) Install Python 3.13
      - Download from python.org
      - Add Python to PATH during installation
   
   b) Install Dependencies
      $ cd d:\FYIUploader
      $ pip install -r requirements.txt
   
   c) Configure OAuth Credentials
      - Set up Facebook Developer Account
      - Create Instagram Business Account connection
      - Generate YouTube API credentials
      - Add credentials to config.py
   
   d) Run the Application
      $ python main.py
      
      First run will create:
      - SQLite database (accounts.db)
      - Configuration files
      - Log directories


2. INITIAL CONFIGURATION
   ─────────────────────────────────────────────────────────────────────────
   
   When you first launch:
   
   a) Security Setup
      - Create admin account
      - Set strong password
      - Enable 2-Factor Authentication (recommended)
   
   b) Connect Social Accounts
      - Navigate to Platforms tab
      - Click "Connect Account" for each platform
      - Follow OAuth authorization flow
      - Confirm scopes are granted
   
   c) Team Setup (Optional)
      - Invite team members via Team Collaboration tab
      - Assign roles: Admin, Editor, Viewer
      - Configure permissions for each role
   
   d) Branding (Optional for White-label)
      - Navigate to White-label Settings
      - Upload company logo
      - Customize color scheme
      - Set custom domain

COMMON WORKFLOWS
================================================================================

WORKFLOW 1: SCHEDULE A MULTI-PLATFORM POST
────────────────────────────────────────────────────────────────────────────

Step-by-step:

1. Click Calendar tab
2. Double-click desired date
3. Enter post content
4. Click "Add Media" to upload image/video
5. Click "Select Platforms" checkbox for:
   ☐ Facebook
   ☐ Instagram
   ☐ Twitter
   ☐ LinkedIn
6. Select time to post
7. Enable "Cross-post" option
8. Click "Preview All"
9. Review on each platform
10. Click "Schedule Post"

Result: Post automatically publishes across all selected platforms at
        specified time.


WORKFLOW 2: CREATE CONTENT & SCHEDULE IN BULK
────────────────────────────────────────────────────────────────────────────

Step-by-step:

1. Prepare 10-50 videos in a folder
2. Click Bulk Upload tab
3. Click "Select Files"
4. Choose all video files
5. For each video:
   - Add description (copy from spreadsheet)
   - Select platforms
   - Set schedule dates (auto-increments)
   - Add hashtags (pull from Hashtag Tools)
6. Click "Start Upload"
7. Monitor progress
8. Check completion report

Result: All videos scheduled across weeks - hands-free publishing!


WORKFLOW 3: MANAGE TEAM APPROVAL WORKFLOW
────────────────────────────────────────────────────────────────────────────

Step-by-step:

1. Add team members in Team Collaboration tab
2. Create post in Calendar
3. Click "Send for Approval"
4. Select approver from dropdown
5. Add approval comments/notes
6. Approver receives notification
7. Approver clicks post notification
8. Approver reviews content
9. Approver clicks "Approve" or "Reject"
10. If approved: automatically publishes at scheduled time
11. If rejected: creator gets notification with feedback

Result: Professional content review workflow established!


WORKFLOW 4: OPTIMIZE CONTENT WITH AI
────────────────────────────────────────────────────────────────────────────

Step-by-step:

1. Go to AI Generator tab
2. Enter post topic
3. Generate captions (5-10 variations)
4. Select best caption option
5. Go to Hashtag Tools tab
6. Generate 20-30 hashtag suggestions
7. Select relevant hashtags
8. Save as collection
9. Go to Calendar tab
10. Create new post
11. Use generated caption
12. Use saved hashtag collection
13. Schedule post

Result: AI-optimized post ready for maximum engagement!


WORKFLOW 5: MONITOR PERFORMANCE & RESPOND QUICKLY
────────────────────────────────────────────────────────────────────────────

Step-by-step:

1. Click Monitoring tab
2. Enable alerts for:
   - High engagement posts (>100 interactions)
   - Brand mentions
   - Comments needing response
3. Keep window open or check periodically
4. When alert triggers:
   - See notification popup
   - Click to view full context
   - Go to Social Inbox tab
   - Reply to message immediately
5. Track response time metrics
6. Review conversations in audit log

Result: Real-time engagement with your audience!

TROUBLESHOOTING COMMON ISSUES
────────────────────────────────────────────────────────────────────────────

Issue: Instagram Upload Fails with Permission Error
Solution:
1. Check Instagram OAuth scopes in Security tab
2. Verify "instagram_content_publish" scope is granted
3. Review OAUTH_SCOPES_FIX.md for detailed steps
4. Reconnect Instagram account

Issue: Scheduled Post Not Publishing
Solution:
1. Verify time zone is correct in Settings
2. Check internet connection
3. Verify platform account is still authorized
4. Check Social Inbox for error messages
5. Manually retry publishing

Issue: Low Engagement on Posts
Solution:
1. Check posting time using Analytics
2. Use AI Generator for better captions
3. Add relevant hashtags using Hashtag Tools
4. Engage with similar accounts
5. Try different content types

Issue: Slow Application Performance
Solution:
1. Clear browser cache
2. Reduce number of open tabs
3. Restart application
4. Check internet connection
5. Update to latest version

FREQUENTLY ASKED QUESTIONS
================================================================================

Q: Can I use the same account for multiple users?
A: No, use Team Collaboration to invite team members. Each gets unique login.

Q: What's the maximum number of scheduled posts?
A: Unlimited. The system can handle thousands of scheduled posts.

Q: Can I edit a post after it's published?
A: Yes, in Social Inbox, but Facebook/Instagram will show it was edited.

Q: How long does it take to post across multiple platforms?
A: Usually 30 seconds. Check Real-time Monitoring for status.

Q: Can I schedule posts from mobile?
A: Not yet, but you can view analytics and respond to messages on mobile.

Q: What happens if the scheduled time passes?
A: Post publishes immediately when you reconnect to internet.

Q: How do I delete my account?
A: Contact admin. Data can be exported before deletion.

Q: Can I revert to a previous version of a post?
A: Yes, Content Library stores version history for 90 days.

Q: Is there a phone app?
A: Mobile app coming in Phase 5. Use web interface for now.

Q: How often are analytics updated?
A: Real-time metrics refresh every 5-10 minutes.
