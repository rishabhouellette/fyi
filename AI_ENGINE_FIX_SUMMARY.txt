PROBLEM IDENTIFIED
================================================================================

ERROR MESSAGE:
```
ERROR | ai_engine | Caption generation error: DatabaseManager.log_activity() 
       got an unexpected keyword argument 'target_type'
ERROR | ai_engine | Hashtag generation error: DatabaseManager.log_activity() 
       got an unexpected keyword argument 'target_type'
```

ROOT CAUSE:
The ai_engine.py was calling log_activity() with wrong parameter names:
- Used: target_type, target_id, metadata
- Expected: resource_type, resource_id, details

DATABASE METHOD SIGNATURE (correct):
```python
def log_activity(self, team_id: int, user_id: int, action: str,
                resource_type: str = None, resource_id: int = None,
                details: Dict = None):
```

================================================================================
WHAT WAS FIXED
================================================================================

File: ai_engine.py

Fixed 4 locations where log_activity() was called incorrectly:

1. Caption Generation (line ~80)
   Before: target_type="ai_engine", target_id=None, metadata={...}
   After:  resource_type="ai_engine", resource_id=None, details={...}

2. Hashtag Generation (line ~160)
   Before: target_type="ai_engine", target_id=None, metadata={...}
   After:  resource_type="ai_engine", resource_id=None, details={...}

3. Best Time Prediction (line ~235)
   Before: target_type="ai_engine", target_id=None, metadata={...}
   After:  resource_type="ai_engine", resource_id=None, details={...}

4. Content Recommendations (line ~365)
   Before: target_type="ai_engine", target_id=None, metadata={...}
   After:  resource_type="ai_engine", resource_id=None, details={...}

PARAMETER MAPPING:
├─ target_type → resource_type
├─ target_id → resource_id
└─ metadata → details
